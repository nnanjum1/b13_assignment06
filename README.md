# DigiTools

## Description
DigiTools is a modern digital tools platform that provides users with premium AI-powered utilities, design resources, and productivity tools in one place. It helps creators and professionals work faster and smarter with an easy-to-use interface.


## 🛠️ Technologies Used
- React.js 
- Tailwind CSS 
- JavaScript
- vite
- React Toastify 


##  Features

-  **Smart Cart System**  
  Add, remove, and manage products easily with real-time updates.

-  **Fast & Responsive UI**  
  Fully responsive design optimized for mobile, tablet, and desktop.

-  **Modern Product Showcase**  
  Beautiful product cards with badges, pricing, and feature lists.
